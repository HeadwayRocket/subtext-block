== Installation ==

Follow the steps below or watch our video [tutorial](http://headwayrocket.com/videos/how-to-install-a-headway-block)

1. Download the plugin from your Headway Dashboard
2. In your WordPress dashboard go to Plugins > Add New > Upload
3. Drag to downloaded zip file and click on Upload
4. Once uploaded, activate the plugin

= Using the block in Headway =

Once you have installed the plugin, follow these steps:

1. Go to Headway Visual Editor in Grid mode
2. Click on the page where your want your Subtext block to be
3. Draw a block and select "Subtext"
